# **App Name**: Smart Digital Campus

## Core Features:

- House System & Skill Tree: Gamified system with real-time leaderboards to track student achievements and skill progression.
- Real-Time Collaboration Workspace: Digital workspace for project groups with co-editing, whiteboard, and integrated video chat.
- Digital Well-being Hub: Portal for students to check-in their mood, access counseling, and mental health resources.
- Interactive Parent Portal: Real-time dashboard for parents with attendance, grades, announcements, and direct chat with teachers.
- Adaptive Learning with AI Tutor: AI tutor which adjusts the learning path using diagnostic quizzes and offers assistance using simpler examples as a tool.
- Real-time data synchronization: Data synchronized across all clients and devices instantly using Firebase.
- PPDB Application: Handle the alur of student onboarding to school with email verifications triggered when a document is written to a Firebase collection.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a modern and intellectual feel, representing knowledge and trust.
- Background color: Light gray (#F5F5F5), very lightly tinted toward indigo, to provide a clean and calming backdrop that enhances focus and readability.
- Accent color: Soft lavender (#C5CAE9), providing a gentle contrast and highlighting important interactive elements without overwhelming the user.
- Headline font: 'Space Grotesk' sans-serif font for a computerized feel, well-suited to titles and headers.
- Body font: 'Inter' sans-serif for clean readability.
- Consistent and modern icon set aligned with the educational context.
- Clean and intuitive layout with a focus on user experience.
- Subtle and meaningful animations for transitions and interactions.
